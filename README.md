# Tecnologias utilizadas
## Back end
- Java 8
- Spring Boot
- JPA / Hibernate
- Postgres
## Front end
- HTML / CSS / JS / TypeScript
- ReactJS
- Routes
- Axios
- Maps
## Deploy
- Back end: Heroku
- Front end web: Netlify
